Operating systems

Linux

I use two different Linux distributions. On my main PC, I enjoy gaming in my free time, so I use Linux Mint 20.1. It suits my use case and I love the freedom and flexibility Linux provides, as well as the community built around it.

On my laptop, which is primarily for work, I use Artix Linux. Artix is essentially Arch without systemd. I don't hate systemd, I just prefer runit, though I have not tried OpenRC or s6 which are two other init systems offered in a base install of Artix.

Really, you can accomplish anything with any Linux distribution, with enough time and patience. I would tend to recommend Arch-based distros above all others, though I suppose personal preference trumps all.

Windows

I have a Windows dual-boot on my laptop and PC for the occasions where I need to use software such as .NET Framework which is not supported on Linux.

Utilities

Suckless tools

I use DWM (dynamic window manager) as my window manager of choice on my laptop. DWM is infinitely customisable by its very nature. You are intended to overwrite, modify, and extend the source code to fit your individual needs, or install extra features others have already made for you. All Suckless offer is the base code and an empty palette for you to work on.

My terminal emulator of choice is ST (simple terminal). Like DWM, it is made by Suckless and is also offered as a modifiable base upon which to build your own release.

Web browser

Brave, I believe, is the best web browser out there at the moment. It's fast, contains almost every privacy feature after changing a couple of settings, supports web extensions for Chrome, and is fully open-source. Brave strikes the perfect balance between solid privacy, great speed, and ease-of-use.

Text editors

For all-purpose text editing and automation I use Vim. Vim is great. It's lightweight, powerful, terminal based, and is very efficient when you learn to use it.

For coding projects that don't require a full IDE, VSCodium is my go-to. It's a totally open-source fork of VS Code with better privacy. Awesome! VS Code is open-source, but the binary download available on MS' website is closed-source, and contains Microsoft's telemetry (usage data collection).

Lastly, my IDE of choice is JetBrains Rider. It, unfortunately, isn't open-source. It also isn't free, but it is available on Linux, works very well, and I get it for free as a student. If a fully-featured, free and open-source .NET IDE becomes available, I will likely switch to it.

Multimedia

The single best image viewer on Linux is SXIV (Simple X Image Viewer). It's feature-rich, extremely lightweight, very fast, and fully open-source.

There are lots of options for video players, but the one I use is mpv. Terminal-based, fast, open-source. I've always had technical issues with VLC, and I have no need for the advanced features, so I don't use it.

My music player of choice is MOC (Music On Console). It has a tonne of useful features, namely the ability to detach the player from the server while keeping it playing in the background, play from URL, create playlists, show lyrics, and much more, all in a console-based application that runs very quickly.

Creative production

I feel like GIMP (GNU Image Manipulation Program) needs no introduction as the free, open-source, cross-platform version of Photoshop. I use it for most editing projects, but I also use ImageMagick for quick changes like image format conversions.

The best Linux video editor is Kdenlive, without a doubt. It's FOSS, fast, and powerful.

I've used Audacity for years and it has always done a great job for me.
